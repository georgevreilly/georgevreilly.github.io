'https://www.example.com/2016/07/12/release/index.html' 'https://www.example.com/pre/fix' + '2016/07/12/release/index.html' -&gt; 'https://www.example.com/2016/07/12/release/index.html' </pre> <p>The last one is a problem, as the path after the hostname, <tt class="docutils literal">/pre/fix</tt>, is lost.</p> <p>We can fix it using <a class="reference external" href="https://docs.python.org/2/library/urlparse.html#urlparse.urlparse">urlparse</a> to separate that path from the hostname:</p> <pre class="brush: python"> from six.moves.urllib_parse import urljoin as abs_urljoin, urlparse def urljoin(site, path): return abs_urljoin(site, path_urljoin(urlparse(site).path, path)) </pre> <p>which yields:</p> <pre class="literal-block"> 'https://www.example.com' + 'foo/bar/quux.js' -&gt;
