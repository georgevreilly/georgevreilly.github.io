On Windows, <tt class="docutils literal">os.path.join</tt> is an alias for <tt class="docutils literal">ntpath.join</tt>, which always uses <tt class="docutils literal">\</tt>. When dealing with URLs, we always want forward slashes, regardless of platform, so <tt class="docutils literal">posixpath.join</tt> should be used to build URL paths.</p> <p>Running:</p> <pre class="brush: python"> from __future__ import print_function from six.moves.urllib_parse import urljoin as abs_urljoin from posixpath import join as path_urljoin def urljoin(site, path): return abs_urljoin(site, path) def test_join(site, path): result = urljoin(site, path) print("'{0}' + '{1}'\n\t-> '{2}'".format(site, path, result)) return result local_path = path_urljoin('2016', '07', '12', 'release', 'index.html') test_join('https://www.example.com', 'foo/bar/quux.js') test_join('https://www.example.com', local_path) test_join('https://www.example.com/', local_path) test_join('https://www.example.com/prefix', local_path) </pre> <p>Yields:</p> <pre class="literal-block"> 'https://www.example.com' + 'foo/bar/quux.js' -&gt; 'https://www.example.com/foo/bar/quux.js' 'https://www.example.com' + '2016/07/12/release/index.html' -&gt; 'https://www.example.com/2016/07/12/release/index.html' 'https://www.example.com/' + '2016/07/12/release/index.html' -&gt; 'https://www.example.com/2016/07/12/release/index.html' 'https://www.example.com/pre/fix' + '2016/07/12/release/index.html' -&gt; 'https://www.example.com/2016/07/12/release/index.html' </pre> <p>The last one is a problem, as the path after the hostname, <tt class="docutils literal">/pre/fix</tt>, is lost.</p> <p>We can fix it using <a class="reference external" href="https://docs.python.org/2/library/urlparse.html#urlparse.urlparse">urlparse</a> to separate that path from the hostname:</p> <pre class="brush: python"> from six.moves.urllib_parse import urljoin as abs_urljoin, urlparse def urljoin(site, path): return abs_urljoin(site, path_urljoin(urlparse(site).path, path)) </pre> <p>which yields:</p> <pre class="literal-block"> 'https://www.example.com' + 'foo/bar/quux.js' -&gt; 'https://www.example.com/2016/07/12/release/index.html' 'https://www.example.com/pre/fix' + '2016/07/12/release/index.html' -&gt; 'https://www.example.com/pre/fix/2016/07/12/release/index.html' </pre> <p>Note that if the <tt class="docutils literal">path</tt> argument to my <tt class="docutils literal">urljoin</tt> function is absolute, any path suffix to <tt class="docutils literal">site</tt> is discarded:</p> <pre class="literal-block"> 'https://www.example.com/pre/fix' + '/abs/path' -&gt;
