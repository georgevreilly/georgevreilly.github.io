Vim: expressions in substitutions

There were a number of links in the HTML from my original blog that looked like http://mydomain/?url=http%3a%2f%2fwww.seat­tle­opera.org%2ftick­et­s%2fring%2f (for referral tracking).

This Vim sub­sti­tu­tion to the rescue:

:s/%\([0-9a0f][0-9a-f]\)/\= nr2char(str2nr('0x'.submatch(1), 16))/g
  • Match all fragments that look like % HEX HEX.
  • Capture the two hex digits, \1 or submatch(1)
  • Convert the match into a number, base 16: str2nr('0x'.submatch(1), 16)
  • Convert that number into a character: nr2char
  • \= in a substitute string means treat the rest as a VimL expression.
